## 🛠️ Stack Tecnológico

```
🧠 Anthropic (Claude)        → 100% da inteligência
   • Opus 4.6                → Orquestração + estratégia
   • Sonnet 4.6              → Copy + análise visual
   • Haiku 4.5               → DM em tempo real

🎨 Google Vertex AI          → Toda mídia visual
   • Imagen 4                → Imagens (80% dos casos)
   • Veo 3                   → Vídeos state-of-art
   • Chirp 3                 → Voz multilíngue
   • Speech-to-Text v2       → Transcrição padrão

🔗 Meta + Zernio             → Distribuição
   • Meta Ads API            → Campanhas
   • Zernio                  → Publicação multi-rede

🔄 Fallbacks Especializados  → Casos específicos
   • Ideogram v2             → Text-in-image complexo
   • ElevenLabs Scribe v2    → Transcrição PT-BR premium
```
